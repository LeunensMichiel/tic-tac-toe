# Tic Tac Toe

## Run this app

1. Install Depedencies

```bash
yarn

```

2. Run dev preview

```bash
yarn dev

```

> App will be running on http://localhost:3000/

or

2. Build

```bash
yarn build && yarn preview

```

> App will be running on http://localhost:4173/

